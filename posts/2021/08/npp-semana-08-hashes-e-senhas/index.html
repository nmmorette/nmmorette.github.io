1. Caso.zip

Alguns fornecedores informam o checksum os executáveis no site, porém não encontramos no caso do Zoom. Foi necessário fazer o download no site oficial e checar:

Para esse lab, fiz um script simples para listagem dos arquivos com seus respectivos hahes, bastou verificar qual hash era o diferente para identificar o arquivo modificado:

Script:

#!/bin/bash
#Script para checagem de hash

#Scritp para verificação do hash md5 em um arquivo na pasta ./original e comparação com arquivos na pasta ./files

#cores

export GREEN='\033[1;32m'
export RED='\033[0;31m'
export PURPLE='\e[35m'
export blink='\e[5m'
export NC='\033[0m' # No Color


#Tratamento do arquivo original 

	ORIGINAL=$(ls original)
	PWDORIGINAL=$(pwd)
	MD5original=$(md5sum  "$PWDORIGINAL"/original/"$ORIGINAL")
	MD5FILTRADO=$(echo "$MD5original" | cut -d " " -f 1)


echo -e ${GREEN}"============================================================================================================================"
echo -e ${blink}"HASH MD5 do arquivo original:"${NC}   ${PURPLE} "$MD5FILTRADO"
echo -e ${GREEN}"============================================================================================================================"${NC}



 
#declando local atual
PWD=$(pwd)
#Lendo arquivos da pasta files
CHECK=($(ls files))
#imprimindo caminho completo dos arquivos
ALL=$(printf '%s\n' ${CHECK[@]/#/$PWD/files/})

#Fazendo a checagem do MD5

	MD5=$(md5sum ${ALL[@]})
	
	ARQUIVOALTERADO=$(printf "${MD5}" | grep -v "$MD5FILTRADO")
	echo -e ${RED} "$ARQUIVOALTERADO" ${NC}

7. Hash

Seguindo a lógica da questão, vamos criar um script para converter toda wordlist em md5, depois base64 e por fim sha1. Assim podemos buscar na conversão o valor informado e verificar em qual linha está o mesmo valor na wordlist original.

Script:

#!/bin/bash

for iq in $(cat password.lst);
do
	echo -n  $iq | md5sum | cut -d "-" -f1; done > md5a.txt

	for ip in $(cat md5a.txt);
	do
		echo -n $ip | base64; done > bs64a.txt
	
		for ic in $(cat bs64a.txt);
		do
			echo -n $ic | sha1sum | cut -d "-" -f1; done > shauma.txt
